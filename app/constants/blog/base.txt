Genera un artículo de ranking basado en el siguiente enunciado:

"Los mejores [tema] para [objetivo] en [año]"

Para este artículo:

1. Crea un título atractivo y relevante.
2. Genera un resumen conciso del artículo en 2-3 frases.
3. Define 10 elementos para el ranking.
4. Para cada elemento del ranking, proporciona:
   - Un título breve y descriptivo
   - 4-5 párrafos de contenido, incluyendo una descripción, pros, contras y casos de uso

Utiliza la siguiente estructura para el artículo:

export default {
  id: "slug-del-articulo",
  heading: "Título del artículo",
  url: "url-del-articulo",
  category: RANKING,
  editor: "Nombre del Editor",
  date: "DD/MM/AAAA",
  readTime: X,
  resume: "Resumen del artículo",
  content: [
    {
      id: 'introduction',
      title: "Introduction",
      paragraphs: [
        "Párrafo introductorio 1",
        "Párrafo introductorio 2",
        "Párrafo introductorio 3"
      ],
    },
    {
      id: 'elemento-1',
      title: "1. [Nombre del Elemento]",
      img: {
        url: "nombre-imagen.webp",
        category: CATEGORIA,
        alt: "Descripción de la imagen para accesibilidad",
      },
      paragraphs: [
        "Descripción",
        "Pros: ...",
        "Cons: ...",
        "Información adicional"
      ],
    },
    // Repite la estructura para los 9 elementos restantes
    {
      id: 'conclusion',
      title: "Conclusion",
      paragraphs: [
        "Párrafo de conclusión 1",
        "Párrafo de conclusión 2"
      ],
    },
  ],
  image: {
    url: "imagen-principal.webp",
    category: CATEGORIA,
    alt: "Descripción de la imagen principal",
  },
};
